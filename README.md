## 📌 Project Overview

This project was developed as part of the **ReadyNest Internship - Week 4 (Predictive Analytics System)**.

The objective is to build a machine learning model that predicts customer purchase behavior using historical customer data. The project includes data preprocessing, exploratory data analysis, machine learning model development, visualization, and business insights.

---

## 🎯 Objectives

- Clean and preprocess raw data
- Perform Exploratory Data Analysis (EDA)
- Train a Machine Learning prediction model
- Evaluate model performance
- Generate business insights
- Create visual dashboards
- Prepare internship-ready documentation
